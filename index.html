About The Thesis Application

In an era where digital communication is paramount, the security of sensitive information for law enforcement agencies like the PNP-Irawan is critical. Standard messaging platforms often present vulnerabilities, risking data breaches and compromising operations.

The PNP-Irawan Secure Messenger is a capstone project developed to address these challenges head-on. It's a dedicated messaging system built with cutting-edge security protocols to protect every piece of information exchanged by personnel.

This thesis, titled "A SECURE MESSAGING SYSTEM USING AES AND PBKDF2 WITH TEMPORARY DATA HANDLING FOR PNP-IRAWAN", was developed by Jhon Luke Q. Cutay, Lawrence Adrian A. Sabenacio, and Jamaica Ruth G. Zabanal, under the advisement of Francis A. Alterado, for Fullbright College.

Core Security Features

Ironclad Encryption

Utilizes AES-256 and PBKDF2 for military-grade security of your messages and files, encrypted on a per-conversation basis.

Temporary Data Handling

Messages are stored locally on user devices and automatically deleted from servers post-delivery, minimizing server-side risk.

Multi-Factor & Adaptive Security

Robust authentication with password or fingerprint biometrics, plus an bypass mechanism that balances security with user convenience via customizable security levels.

Per-Conversation Control

Each conversation can have unique security settings, ensuring granular control over sensitive communications.

User-Focused Design

An intuitive interface designed for ease of use by all PNP personnel, ensuring quick adoption and efficient communication.

Reliable Backend

Powered by Google Firebase for secure authentication, real-time notifications, and message routing.
